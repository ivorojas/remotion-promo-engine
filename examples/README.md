# Examples

Real production output, built with this engine.

## Umbral — launch promo (English)

`umbral-tarot-promo-en.mp4` — the full ~95-second launch video for
**[Umbral](https://umbralcosmos.com)**, a bilingual tarot / cartomancy /
astrology app. Click the file above to open GitHub's video player.

100% authored in code with this engine:
- 14 scenes: hero → ritual → card reveal → three-card spread + a montage of
  every spread shape → the reading (message written on screen) → decks →
  cartomancy → the reader's table (a cursor deals / flips / drags cards) →
  oracle → card of the day → astrology (+ a 12-sign carousel) → learn (course
  modules + quiz) → the altar in real use → a clean outro with CTA.
- Real card art from the app (nothing invented).
- A deep, warm ElevenLabs voiceover (built bilingual — EN + neutral Spanish).
- Music and every SFX **synthesized from pure math** (`scripts/gen-sfx.mjs`) —
  zero third-party copyright.
- Mixed with ducking under the voice, mastered to −15.5 LUFS.

The `preview-*.gif` files are short highlight clips used in the top-level
README (a committed `.mp4` doesn't play inline in a GitHub README — GIFs do;
the full file plays when you open it directly, and links from the README).
