# CLAUDE.md

This project is an **agent-first Remotion promo-video engine**.

Everything you need is in two files — read them before doing anything:

1. **AGENTS.md** — what this is, how to install it into a user's project,
   and the step-by-step workflow (harvest brand → shot list → scenes →
   verify by stills → audio → render → master → version in `releases/`).
2. **METHOD.md** — the craft rules (motion, composition, audio mixing, voice
   direction, honesty of copy, verification discipline). Follow them; they
   are the quality bar.

Then skim the header comments of `src/lib/*` (each primitive documents its
own API and the lessons baked into it) and study `src/scenes/Demo.tsx` as the
reference scene before writing your own.
